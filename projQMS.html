Clearview Dental QMS
This is a second-year Java project developed using Apache NetBeans as part of the core subject "Data Structures and Algorithms." The system features a user-friendly interface designed to improve accessibility and efficiency. It aims to replace the traditional file-based record-keeping methods commonly used in clinics, offering a more organized, reliable, and scalable digital solution for managing patient information and clinic operations.
